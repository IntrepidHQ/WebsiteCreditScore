1. Accounts to create

Sign up for each of these. Use the same billing address across providers to simplify reconciliation.
  • Create a project. Note the project ref from the URL — you'll need it for OAuth callback URLs.
  • Complete the account activation form (legal name, address, tax info). Enable both test and live mode.
  • Create a project, enable Google+ API / Identity API, create an OAuth 2.0 client.
  • Create an API key in Google Cloud Console with PageSpeed Insights API enabled. Raises anonymous rate limits.

3. Domain + DNS

Point the root domain at Vercel and verify email sender identity with Resend.
  • Vercel Dashboard → Project → Settings → Domains. Follow the exact records Vercel shows for your registrar.
  • TXT @ "v=spf1 include:_spf.resend.com ~all"
